    <td><b>Fast Gaussian processes via hierarchical matrix factorizations</b>
    <details>
<summary> Description </summary>

My master's thesis is about accelerating Gaussian process regression, classification, and hyperparameter MLE via randomized methods. Some questions that I am currently trying to answer in my thesis include

<ul>
 <li>How do certain hierarchical factorizations perform on covariance matrices generated by various classes of kernels?</li> 
<li>Efficient ways to evaluate the log-likelihood and its gradient for the MLE of kernel hyperparameters in higher dimensional data (d>2). </li>
<li>In particular, the analysis of various randomized black-box trace estimators. </li>
<li>The optimal length of a truncated eigenfunction expansion of a Gaussian process, given system size, and input dimension. </li>
<li>Which randomized low-rank approximation methods are effective for compressing blocks of Hierarchical matrices? </li>
<li>What types of randomized tools can be used for Gaussian processes associated with general kernels? </li>
<li>How can an ill-conditioned Hessian of a log-likelihood be modified for large-scale second-order MLE? </li>
</details>
